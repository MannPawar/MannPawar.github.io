# Mann Pawar, Portfolio

Personal portfolio of game AI and computational-modeling projects: agent-based
simulation, optimization, and machine learning applied to game worlds and player behavior.

Live site: https://mannpawar.github.io

## Projects
- **Generative Opinion Dynamics in Skyrim**: stochastic agent-based model of 1,010 NPCs with an SLM dialogue layer.
- **Stochastic Valorant Shotcaller**: MILP + XGBoost + MDP team-economy optimizer (live demo).
- **Skyrim Class Optimization**: LP/MILP model of character-build preferences with held-out evaluation.
- **The Silent Killers of Player Retention**: churn intelligence across Steam's Top 50 live-service titles.

Built as a static site (`index.html`).
